# Grazioso Salvare Rescue Animal System

## Overview

This Java application was developed for **IT-145: Foundation in Application Development** at Southern New Hampshire University. The project simulates a rescue animal management system used by Grazioso Salvare to track dogs and monkeys available for rescue missions.

The application demonstrates object-oriented programming principles by using inheritance, encapsulation, constructors, and collections to manage rescue animal information.

---

## Project Objectives

* Design reusable Java classes
* Implement inheritance and encapsulation
* Manage collections of objects
* Create a menu-driven application
* Apply software engineering best practices

---

## Features

* Add new rescue animals
* Reserve animals for service
* Display available animals
* Search by animal type
* Store animal information
* Validate user input
